# alexandra-furlong.github.io

# Workflow Steps
1. Type "npm run dev" in powershell on vs code
2. Verify changes locally by opening a live server then coding 
3. After all changes have been made, stop the dev server with Ctrl+C
4. Type "npm run build" to update output.css with production-ready CSS
5. Add the changes and push to Git

# Notes 
- You always need tor un npm run build BEFORE deploying or pushing code to GitHub if the CSS has changed
- During active development, you can stick with npm run dev for convenience
